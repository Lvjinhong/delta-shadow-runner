Delta External Vision Runner 开发进度
更新时间：2026-07-16
GitHub：https://github.com/Lvjinhong/delta-shadow-runner

[最终目标]
- 交付纯 Windows Python 脚本：桌面截图 → 视觉识别/定位 → 固定路线规划 → 标准键鼠输入 → 画面反馈与卡住恢复。
- 首个游戏 MVP 限定一个固定分辨率、一个地图区域、一个出生区域和一条固定路线。
- 不读取目标进程内存，不注入 DLL，不使用驱动，不修改游戏文件，不实现反作弊规避。
- 不实现敌人识别、自动瞄准、自动射击或战斗决策。

[2026-07-16 当前检查点]
- 早期 React/Node 模拟平台、REST/WebSocket、浏览器 Dashboard、npm/Vite/Playwright 工具链和旧启动入口已从当前工作树移除；历史仍可从 Git 找回。
- 删除前已把旧 TypeScript A* 的关键安全回归迁到 Python：未知节点、贪心陷阱、极端坐标、累计代价溢出、零权边、重复边、悬空边、非有限值和输入不可变。

[验收边界]
- 合成图、受控 Tk 窗口和离线回放只能证明管线行为，不能证明《三角洲行动》准确率。
- 实际准确率只能来自按完整 run 隔离的真实游戏 blind test；必须同时报告样本量、会话数、precision、recall、F1、false lock 和完整路线成功率。
- 如果标准桌面截图或 SendInput 被目标拒绝，记录为不支持，不升级到注入、驱动或规避方案。
